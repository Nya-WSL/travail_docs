- **开始**：开始倒计时，未连接弹幕服务器时按钮不可用
- **暂停**：暂停倒计时，倒计时未开始时按钮不可用
- **继续**：继续倒计时，倒计时未暂停时按钮不可用
- **停止/重置**：停止/重置倒计时，倒计时未开始时按钮不可用，停止后会清空时间，重置只会在倒计时未清零且程序重新启动时显示
    - 加班姬会尝试在程序报错、崩溃和退出时自动保存倒计时数据，在程序重新启动后会自动加载倒计时数据，如果需要手动保存倒计时数据，请使用**保存**按钮
- **增加/减少**：手动增加/减少倒计时时间，倒计时未开始时按钮不可用，可在倒计时运行时使用（这算是给主播准备的小手段吗x
- **保存/读取**：保存/读取倒计时数据，会将当前倒计时保存下来，读取时会覆盖当前倒计时，可在倒计时运行时使用
    - 该功能可能会缓解加班姬因为端口占用导致的无法多开问题
- **礼物设置**：打开礼物设置界面，设置自定义礼物的暴击倍率；设置礼物、数量和时长的规则
    - **加班设置**：倒计时礼物设置
    - **挑战设置**：投喂挑战礼物设置
        - 投喂挑战指赠送礼物可以增加/减少挑战数量，比如赠送礼物可以增加/减少主播的蹲起次数；再比如击杀数挑战等玩法
    - **更新礼物**：手动更新礼物数据
- **颜色设置**：设置倒计时、按钮、文字的颜色
- **统计相关**：加班姬运行期间盲盒的盈亏和收到的礼物统计，启动加班姬时清零
- **界面预览**：在浏览器查看倒计时和挑战的界面，在OBS浏览器源捕获时不建议使用，可能会导致OBS不会刷新界面，如不刷新可在OBS的浏览器源中点击刷新当前页面缓存
- **登录账号**：目前仅跳转至获取身份码的页面，需要手动输入身份码
- **检查更新**：手动检查更新，程序会在启动时自动检查一次更新
- **更新日志**：查看更新日志
- **上传日志**：上传日志到服务器，用于调试和反馈问题，上传日志只会上传加班姬本次运行产生的日志，不会上传历史日志；在上传日志前请确保日志中没有包含敏感数据
